Sensors for the web

A fun little project with Generic Sensor API. Open this application on devices with built-in sensors like a mobile, tablet, or even in your car dashboard to access its sensor's data.
Note: Support for accessing sensors depends on browsers.

 
 

Privacy

This application entirely runs locally on the client and can even run without the internet. The sensor's data aren't stored or transferred externally.

Specifications

Generic Sensor Api Geolocation Sensor Ambient Light Sensor Magnetometer Gyroscope Accelerometer Orientation Sensor
